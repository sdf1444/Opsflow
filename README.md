# OpsFlow

OpsFlow is a production-style internal workflow and approval platform built using React, ASP.NET Core and PostgreSQL.

The application demonstrates engineering patterns commonly found in commercial software, including:

- JWT authentication
- Role-based access control
- Workflow state management
- Audit logging
- Validation
- REST APIs
- Automated testing
- Docker
- Continuous Integration

## Technology Stack

### Frontend

- React
- TypeScript
- Vite
- Material UI
- React Query
- React Hook Form

### Backend

- ASP.NET Core (.NET 8)
- Entity Framework Core
- PostgreSQL

### DevOps

- Docker
- Docker Compose
- GitHub Actions
